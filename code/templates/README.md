# 标准模板文件夹

这个文件夹用于存放模板相关的文档和说明。

## 模板下载

标准模板现在通过系统动态生成，用户可以：

1. 在前端界面点击"下载标准模板"按钮
2. 通过API接口 `/api/exams/template-example/{exam_type}` 获取模板数据
3. 模板会根据考试类型（物理类/历史类）自动生成正确的列结构

## 模板结构（60列）

### 物理类模板包含：
- 基础信息：姓名、学校、班级、年级（4列）
- 必修科目：语文、数学、英语、物理（各4列，共16列）
- 四选二科目：化学、生物、地理、政治（各8列，共32列）
- 总分：原始总分和赋分总分（8列）

### 历史类模板包含：
- 基础信息：姓名、学校、班级、年级（4列）  
- 必修科目：语文、数学、英语、历史（各4列，共16列）
- 四选二科目：化学、生物、地理、政治（各8列，共32列）
- 总分：原始总分和赋分总分（8列）

## 选课组合说明

### 常见组合：
**物理类：**
- 物化生（物理+化学+生物）
- 物化地（物理+化学+地理）
- 物化政（物理+化学+政治）
- 物生地（物理+生物+地理）
- 物生政（物理+生物+政治）
- 物地政（物理+地理+政治）

**历史类：**
- 史化生（历史+化学+生物）
- 史化地（历史+化学+地理）
- 史化政（历史+化学+政治）
- 史生地（历史+生物+地理）
- 史生政（历史+生物+政治）
- 史地政（历史+地理+政治）

### 填写规则：
- 必修科目：必须填写
- 选考科目：根据学生实际选课填写2门，其余2门留空
- 赋分科目：选考科目需要填写原始成绩和赋分成绩
- 总分：需要填写原始总分和赋分总分

## 技术说明

- 模板通过 `TemplateService` 动态生成
- 支持CSV格式下载（可用Excel打开编辑）
- 自动识别考试类型和列映射
- 支持数据验证和错误提示
